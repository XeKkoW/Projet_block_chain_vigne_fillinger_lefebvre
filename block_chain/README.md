Bonjour et bienvenue sur notre projet dédié à la Blockchain.

pour lancer notre solution il vous suffit de vous placer dans le fichier "block-chain" puis de lancer une commande "npm start" dans le terminal

# Ensuite une fois le compte identifié il est possible de remplir les deux champs de textes restants pour identifier un autre compte et une somme d'Ether pour effectuer une transaction directement sur le compte renseigné.

L'intérêt de l'utilisationde la block-chain dans ce projet est de pouvoir effectuer des transctrions de crypto-monaie de façon sécurisé en assurant un anonymat des utilisateurs.
